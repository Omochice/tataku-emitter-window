Options ~
                                              *tataku-emitter-window-options*

This module has some options:

- `cmd` *tataku-emitter-window-option-cmd*

  Command name to open window.
  Default: `'edit'`
- `bufname` *tataku-emitter-window-option-bufname*

  Buffer name.
  Default: `'[scratch]'`
- `filetype` *tataku-emitter-window-option-filetype*

  Filetype
  Default: `''`
